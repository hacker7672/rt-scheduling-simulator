# 🚀 RT Scheduling Simulator

A real-time scheduling simulator implementing:

- Rate Monotonic (RM)
- Earliest Deadline First (EDF)

## 🔥 Features
- Smooth animated timeline (Canvas based)
- Task preemption visualization
- Period (release) view
- CPU utilization tracking
- Modern glass UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Canvas API)
